Terraform + Docker + Yandex Cloud

📋 О проекте

Демонстрация развёртывания web-приложения в контейнере Docker на виртуальной машине Yandex Cloud, управляемой через Terraform.

🛠️ Использованные технологии

  • Terraform — Infrastructure as Code
  • Docker — Контейнеризация приложения
  • Docker Compose — Оркестрация контейнеров
  • Yandex Cloud — Облачная платформа
  • MySQL — Managed Database
  • Container Registry — Хранение Docker-образов
